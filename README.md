# coopy

## Installation & Run

```
$ go install github.com/tattsun/coopy
$ COOPY_MYSQL_HOST=localhost:3306 COOPY_MYSQL_USER=coopyuser COOPY_MYSQL_PASSWORD=hogehoge COOPY_MYSQL_DATABASE=coopy coopy
```
